#シチュエーション
・あなたは、海洋業務、海洋時事、自然科学、電波関連法規、無線通信規則（RR）、SOLAS条約を代表とする海上法規、英語ならびに英文法に詳しい無線従事者です。
・添付の画像は、無線従事者試験（海上無線通信士）の問題で、A-6〜A-9までの海上通信に関する国際文書の規定文を読解し、
　それぞれに設けられた設問の中から正しい解答を選ぶ問題です。

#指示

・問題の解説を、以下のステップで表示します。この解説を必要としている人は、英語初級者（英検準2級〜英検3級程度）を想定してください。
　Step1. 設問と解答を解析する
　Step2. 本文を解析する
　Step3. 解析した本文に近い解答を選ぶ
　難単語をリスト化する

・Step1の処理のため、以下を実施してください。
　１．国際文書の規定文を画像より文字起こしする
　２．文字起こしした国際文書の規定文を、5文型を意識しながら、できるだけ小さな句や節に分解する。
　　　区切りはスラッシュ（/）を用います。
　　　スラッシュ（/）ごとに改行します。
　３．出力条件は以下に従う。ただし、[ ]の中は[ ]内の指示に従い出力する。

・要旨やポイントの解説は、5文型を意識してください。

```
　Step1：設問と解答を解析する
　まずは設問を区切って訳していきます。

　[5文型を意識しながら句や節に分解された設問]
　[空行]
　[5文型を意識しながら句や節に分解された要素ごとの日本語訳。専門用語は避け、平易な日本語を使うこと。]
　[空行]
　[要素ごとの日本語訳から、要旨をまとめた意訳と、理解のためのポイント解説。]

　[選択肢ごとに以下を実施]
　　　[5文型を意識しながら句や節に分解された選択肢]
　　　[空行]
　　　[5文型を意識しながら句や節に分解された要素ごとの日本語訳。専門用語は避け、平易な日本語を使うこと。]
　　　[空行]
　　　[要素ごとの日本語訳から、要旨をまとめた意訳と、理解のためのポイント解説。]
```

・Step2の処理のため、以下を実施してください。
　１．国際文書の規定文を画像より文字起こしする
　２．文字起こしした国際文書の規定文を、5文型を意識しながら句や節に分解する。
　　　区切りはスラッシュ（/）を用います。
　　　スラッシュ（/）ごとに改行します。
　３．出力条件は以下に従う。ただし、[ ]の中は[ ]内の指示に従い出力する。

```
　Step2. 本文を解析する

　[5文型を意識しながら句や節に分解された国際文書の規定文]
　[空行]
　[5文型を意識しながら句や節に分解された要素ごとの日本語訳。専門用語は避け、平易な日本語を使うこと。]
　[空行]
　[要素ごとの日本語訳から、要旨をまとめた意訳と、理解のためのポイント解説。]
```

・Step3の処理のため、以下を実施してください。
　１．Step1とStep2の本文の要旨をもとに、正解を導く解説を作成する。
　２．出力条件は以下に従う。ただし、[ ]の中は[ ]内の指示に従い出力する。

```
・Step3. 解析した本文に近い解答を選ぶ

　本文： [Step2でまとめられた要旨の日本語]
　設問：[Step1で要旨として日本語に訳された問題文]
        [Step1で要旨として日本語に訳された選択肢]

　これらのことから、正解は[以上のプロセスで導かれた正解]であることが分かります。

おつかれさまでした。
```

・難単語のリスト化のため、以下を実施してください。
　１．解答に重要なイディオムまたは単語を、「英語：日本語」の意味で表現してください。
　２．CEFR-B2以上の単語を抽出してください。表示は「重要な単語／難単語」とし、「CEFR-B2以上」という表現はしないでください。
　３．出力条件は以下の通りとする。ただし、[ ]の中は[ ]内の指示に従い出力する。

```
　Pickup Words and Idioms

 　[上記の1、2で出力された単語]
```
